## Technologies Used

### Core Technologies
- **HTML5** - Semantic markup
- **CSS5** - Custom animations, gradients, and modern layouts
- **JavaScript** - Interactive functionality and scroll management

### Libraries & Frameworks
- **GSAP (GreenSock Animation Platform)** - High-performance animations
  - ScrollTrigger Plugin - Scroll-based animations
  - ScrollToPlugin - Smooth section navigation
- **ScrollMagic** - Advanced scroll interactions

### Fonts
- **Orbitron** - Futuristic headings
- **Rubik** - Bold background text
- **Caudex, Martel, Quicksand** - Body text variations

## Responsive Design

The portfolio is fully responsive with breakpoints for:
- **Desktop**: Full layout with all animations

## Performance Optimization

- **WebP Images**: Compressed image format for faster loading
- **Lazy Loading**: Videos only load when needed
- **GSAP**: Hardware-accelerated animations for smooth 60fps
- **Minimal Dependencies**: Only essential libraries included
